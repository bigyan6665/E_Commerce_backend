# E-commerce backend system(in-progress)
Designing a role-based backend system for E-commerce business.

**Features**
---
- Authentication and Role-based access control
- Restful CRUD APIs using JWT
- APIs for authentication, customer and profiles manipulation(till now)
- Schema validation

**Tech Stack**
---
- FastAPI
- SQLite
- SQLAlchemy, Pydantic
- Redis(for blocklisting JWTs), Cloudinary
